## ✨ Overview

**Unburden** is a high-impact, anonymous space designed to facilitate honest conversations about addiction and recovery. Built with a premium "Midnight & Aura" aesthetic, it prioritizes user safety and privacy above all else, allowing individuals to share their journeys without the weight of exposure.

> "A high-impact, anonymous space where recovery begins with honest conversation."

---

## 🚀 Key Features

### 🔒 Total Anonymity
Participate as a guest or use a pseudonym. Your identity is protected by default, giving you the freedom to be truly honest.

### 👥 Cove Circles
Topic-based support spaces for specific addictions and life struggles. Find the community that understands exactly what you're going through.

### 📝 Private Journaling
A local-first, distraction-free space for self-reflection. Your entries stay on your device unless you choose otherwise.

### 🛡️ Safety & Crisis Support
Persistent access to region-specific crisis resources and active community moderation to ensure a safe environment for everyone.

### 🎚️ User Control
Customize your experience with trigger filters, data retention settings (7, 30, or 90 days), and granular privacy toggles.

## 🛠️ Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/SharonSoul/unburden.git
   cd unburden
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Visit the app**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application in action.

## ⚠️ Safety Disclaimer

**Unburden is a peer support platform, not a substitute for professional medical care.**  
If you are in immediate danger or experiencing a crisis, please seek professional help immediately:
- **US:** Call or text [988](https://988lifeline.org/) (Suicide & Crisis Lifeline)
- **Canada:** Call [1-833-456-4566](https://www.talksuicide.ca/)
- **Global:** Contact your local emergency services (e.g., 911).
